About

On this site, I present selected notes from my zk notebook. The notes are loosely categorized using tag keywords and often cross-reference each other in a zettelkasten-like fashion. With regard to their topic, the majority of notes concern themselves with all matters related to computing. This includes—among other things—programs I have written, software I use, and configuration files for that software. The website is generated using Hakyll, a static site generator written in the Haskell programming language. More information about the setup is available in a seperate note.
